### 方式 3：命令行（开发者）

```bash
cd ~/.claude/skills/desktop-pet-generator

python scripts/pet_generator.py \
  --image your-pet.png \
  --name "宠物名" \
  --mode css \
  --deployment html \
  --output ./my-pet
```
